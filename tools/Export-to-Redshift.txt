How to Export Usergrid Data to Redshift
=======================================

Setup a computer with the following software

   Java 7
   Python 2.7
   The Python psychopg2 module

Create a directory /usr/local/redshift_upsert

Place the usergrid-tools-0.0.28-SNAPSHOT.jar in the directory

Place the usergrid-custom.properties file in the directory, edit and set configuration values

Place the redshift_upsert.py script in the directory, edit and set configuration values

Place the script redshift.sh in the directory
